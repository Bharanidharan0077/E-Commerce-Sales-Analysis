# 📊 E-Commerce Sales Analysis Dashboard

## 📌 Project Overview
This project presents an **E-Commerce Sales Analysis Dashboard** built using Power BI.  
The dashboard helps analyze revenue performance, sales trends, product performance, and regional distribution.

It provides insights that help businesses make **data-driven decisions** and understand sales patterns.

## 📊 Dashboard Features

### 1️⃣ Monthly Revenue Trend
Shows how revenue changes across months to identify seasonal trends.

### 2️⃣ Sales Region Map
Displays sales distribution across different regions.

### 3️⃣ Top Products
Highlights the highest revenue-generating products.

### 4️⃣ Revenue by Category
Compares performance across product categories like:
- Accessories
- Electronics
- Mobiles

### 5️⃣ Interactive Filters
Users can filter the dashboard by:
- Month
- Category
- Region
